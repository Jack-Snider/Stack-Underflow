[English]
Read If there is 'readMe.txt' files in the directory

[한국어]
만약 폴더에 'readMe.txt' 파일이 있으면 반드시 읽어볼것
-------------------------------------------------------------

[Gid add/commit/push , pull command]

로컬 -> 원격

1. git add .
2. git commit -m "남길 메세지"
3. git push origin 자기브런치

원격 -> 로컬
1. git pull origin 가져올브런치 
(보통은 main임)
